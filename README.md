# Aplicativo de Login

Este projeto é um aplicativo de login desenvolvido com React Native e Expo.

## Como executar o aplicativo

### Instalar dependências
- npm install

###Abrir o aplicativo

-Pressione A para abrir no Android.

-Pressione W para abrir no navegador.

###Sobre o projeto

-Este aplicativo realiza:

-Cadastro de professor

-Armazenamento local usando AsyncStorage

-Login utilizando os dados cadastrados

-Exibição de tela logada

-Função de logout

-Interface integrada ao estilo visual do site
